IIMS static file check — Express is serving THIS project folder (`public/` from cwd).

1. Open **`http://localhost:<PORT-from-banner>/`** — not an older `:5002` tab if banner says `:5003`.
2. **`GET http://localhost:<PORT>/api/build-info`** JSON includes **`public_index`** (`step1_index_rev`, `styles_css_query_v`). After `git pull` these must match `public/index.html` on disk.

Release stamp: set `IIMS_BUILD_STAMP` in `.env` if you want a custom banner label.

JSON: `GET /api/build-info`

If port fallback messages appear (`Port 5002 is in use`), stop stray Node: `Get-Process node | Stop-Process -Force`, then run `npm start` again.
